# Telegram Support (Unofficial) - API Connection Fixer

This tool helps developers and users troubleshoot connection issues with Telegram Servers. If you are experiencing "Connecting..." status or API timeouts, use this diagnostic tool.

## Key Features:
* MTProto Proxy Status Check
* Telegram Data Center (DC) Connectivity Tests
* Troubleshooting for API login issues
